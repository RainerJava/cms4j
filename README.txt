Custom Tours
=========================================




How to use the Custom Tours website
=================================================

1. Deploy the Webservice archive file ct-website-VERSION.war of the website.
   E.g. for an Apache Tomcat installation, copy the file to the the directory
   TOMCATDIR\webapps.

2. Start the server (e.g. Tomcat).

3. Open the website using the website's endpoint
   (e.g. http://localhost:8080/ct-website-VERSION/)
